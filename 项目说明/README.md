# 项目说明目录

本目录用于记录项目开发过程中的所有问答、技术讨论和解决方案。

## 目录结构

所有记录按日期组织，每天的问题保存在对应的日期文件夹中：

```
项目说明/
├── README.md                    # 本文件（目录说明和索引）
├── AI规则说明.md                # AI 记录规则说明
├── 20260110/                    # 2026年1月10日的记录
│   ├── 01-如何编译x86-64平台镜像.md
│   ├── 02-为什么项目中有个乱码文件夹.md
│   └── 03-删除乱码文件夹.md
├── 20260111/                    # 2026年1月11日的记录（示例）
│   └── 01-xxx.md
└── ...
```

## 文件命名规则

### 日期文件夹
- **格式**: `YYYYMMDD`（8位数字，无分隔符）
- **示例**: `20260110` 表示 2026年1月10日

### 记录文件
- **格式**: `当日提问索引-标题.md`
- **索引**: 从 `01` 开始，同一天内递增
- **标题**: 问题的简短描述（中文）

**示例**:
- `01-如何编译x86-64平台镜像.md` - 当天的第1个问题
- `02-配置网络相关问题.md` - 当天的第2个问题

## 文件内容结构

每个记录文件包含以下部分：

1. **用户问题** - 原始问题
2. **项目信息** - 相关项目背景
3. **问题分析** - 问题分析和理解
4. **执行步骤** - 实际执行的命令和结果
5. **详细解答** - 完整的技术解答和步骤
6. **技术细节** - 相关的技术细节和配置
7. **常见问题** - FAQ 和问题排查
8. **相关文件** - 涉及的文件列表
9. **参考资源** - 相关文档和链接
10. **后续步骤** - 待完成的任务

## 使用说明

- 所有问答记录按日期组织，每天的问题集中在一个文件夹
- 可以通过日期文件夹快速定位特定日期的问题
- 每个文件都是独立的，包含完整的上下文信息
- 建议定期查看，避免重复提问
