# 03: 删除乱码文件夹

## 用户问题

> 删除乱码文件夹

## 问题背景

在项目根目录中发现了一个显示为乱码的文件夹 `椤圭洰璇存槑`（显示为 `Ŀ˵`），这是字符编码问题导致的。

## 执行步骤

### 1. 删除乱码文件夹

使用 PowerShell 命令删除乱码文件夹：

```powershell
Get-ChildItem -Directory | Where-Object { 
    $_.Name -match '[^\x00-\x7F]' -and $_.Name -ne '项目说明' 
} | ForEach-Object { 
    Remove-Item $_.FullName -Recurse -Force -ErrorAction SilentlyContinue
}
```

**执行结果**: ✅ 成功删除乱码文件夹 `椤圭洰璇存槑`

## 问题原因回顾

乱码文件夹是由于字符编码不匹配导致的：
- 正常文件夹 `项目说明` 使用 UTF-8 编码
- 乱码文件夹可能是用 GBK/GB2312 编码创建，被误读为 UTF-8
